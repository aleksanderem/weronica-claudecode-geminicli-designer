O wtyczce Figma auto grafik

Plugin umozliwia komunikacje Gemini CLI / Claude Code z Figma poprzez WebSocket. Rozwiązanie to umozliwia pracę bezpośrednio w projektach gdzie domyslny Figma MCP działa tylko jako tryb do odczytu tylko dla developerów.

Wersja: 1.0.0

Jak tego uzywac?

  1. Upewnij sie, ze MCP jest zainstalowane i dziala
  2. Polacz sie z serwerem poprzez odpowiedni port (domyslny: 3055)
  3. Jak sie polaczysz, skopiuj ID kanału i wklej to do Gemini/CC
  4. Wazne: 1 projekt == 1 kanał ID. Jak chcesz pracować z kilkoma projektami na raz, w kazdym musisz uruchomic plugin
